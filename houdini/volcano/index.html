Houdini Volcanic Eruption

Updated Sept. 28, 2020



Volcanic Eruption created and rendered in Houdini. Lava created using FLIP fluid with divergence attribute and emission / color mapped shader. Smoke effects done with Pyro dynamics and some VEX controllers for the flame emission. Lightning geometry generated with custom lightning tool.
