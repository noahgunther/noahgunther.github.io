Houdini Work




Droplets Tool
SOP-based (no fluid sim) tool for Houdini that dynamically creates droplet geometry and wetmaps on any surface.


Vines Tool
A Houdini tool for procedurally growing vines on any input geometry that creep their way toward the light.


Robot Artist
A virtual robot created in Houdini that procedurally draws 2D images of 3D meshes - and sometimes gets abstract.


Pipe Dream Redux
A procedural animation and randomization project to recreate the classic Microsoft Windows "Pipe Dream" screensaver using VEX and Python.


Volcanic Eruption
A Volcanic Eruption in Houdini using pyro and fluid simulation, and a custom lightning tool created with VEX.


Stained Glass Tool
Tool for Houdini that creates stained glass mosaic geometry from input images.


Interactive Fireworks
A Python state and VEX based Houdini tool for interactively creating animated fireworks in the viewport.
